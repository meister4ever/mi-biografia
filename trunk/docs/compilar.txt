 
Para compilar se corre el shell script runUnitTests.sh
Se compila todo y se corren las pruebas automáticamente
se corre (desde terminal o consola) con:
./runUnitTests

si no les da permiso linux tienen que darselo con:
chmod a+x runUnitTests.sh

después de correr las pruebas crea el xml con los resultados.
Se requiere tener instalado CMake
